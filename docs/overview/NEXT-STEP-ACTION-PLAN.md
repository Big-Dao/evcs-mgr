# EVCS Manager 下一步行动计划（Week 4-8, 2025 P3）

> **版本**: v1.0 | **最后更新**: 2025-11-10 | **维护者**: 项目管理办公室（PMO）  
> **适用范围**: 管理层、交付负责人、跨团队协调

---

## 1. 总体目标

- **阶段目标**: 在 P3 阶段（共 8 周）完成真实支付接入、协议事件闭环、演示数据与指标体系、性能与安全基线。
- **衡量指标**:
  - 真实支付: 支付宝/微信沙箱完成支付、对账、回调验签。
  - 协议链路: OCPP/云快充事件→订单/告警流水打通，监控可见。
  - 演示体验: Dashboard 与前端页面加载真实数据（tenant 1001），无模拟提示。
  - 性能安全: 压测达到 500~2000 TPS；高危漏洞为 0；CI/CD 全绿。
