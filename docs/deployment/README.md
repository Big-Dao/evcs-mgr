## 🎯 使用指南

### 快速部署
1. 参考 [DEPLOYMENT-GUIDE.md](DEPLOYMENT-GUIDE.md) 进行系统部署
2. 使用 [DOCKER-CONFIGURATION-GUIDE.md](DOCKER-CONFIGURATION-GUIDE.md) 配置Docker环境
3. 参考 [DOCKER-OPTIMIZATION.md](DOCKER-OPTIMIZATION.md) 优化部署性能

### 环境要求
- **系统**: Linux/macOS/Windows
- **Docker**: 20.0+
- **Docker Compose**: 2.0+
- **内存**: 最少8GB，推荐16GB
- **磁盘**: 最少20GB可用空间
