## 测试访问指南

使用 **platform_admin_a** 账号登录后，可以在以下页面查看数据：

1. **充电站管理** (`http://localhost:3000/stations`)
   - 可以看到3个充电站
   - 每个站点显示充电桩数量、运营状态等信息

2. **充电桩管理** (`http://localhost:3000/chargers`)
   - 可以看到10个充电桩
   - 包含快充、慢充等不同类型
   - 显示实时状态（运营中、充电中等）

3. **充电订单** (`http://localhost:3000/orders`)
   - 可以看到10个订单
   - 包含已支付、待支付、充电中等不同状态
   - 显示充电量、时长、金额等详细信息

4. **计费方案** (`http://localhost:3000/billing-plans`)
   - 可以看到3个计费方案
   - 每个方案包含分时段的电价和服务费

## 数据特点

1. **多租户隔离**: 所有数据都属于租户ID=100，与其他租户完全隔离
2. **完整业务场景**: 涵盖不同类型的充电站、充电桩和订单状态
3. **真实数据分布**: 包含不同功率的充电桩、不同时段的计费方案
4. **时间维度**: 订单跨越最近10天，便于测试时间筛选功能
5. **状态多样性**: 包含运营中、充电中、已支付、待支付等多种状态

## 注意事项

- 所有数据都使用 `platform_admin_a` 用户（ID: 100001）创建
- 数据生成时间为执行SQL脚本的时间
- 订单的充电时间是相对当前时间的偏移量
- 计费方案覆盖全天24小时，分为谷、平、峰不同时段
